# Data I/O

---

## Positional Args

`name`과 `data`는 keyword 없이 positional 사용 가능:

```python
u.rjb("config")              # name="config"
u.wjb({"k": "v"}, "out")    # data=, name=
u.hjb("config")
```

---

## Key Parameters

```python
data=           # 쓸 데이터 (w/u 액션 필수)
name=           # 파일명 (확장자 자동추가, 미지정=dumps)
keydata=        # 중첩 경로 — "user/email", "items[0]/name"
walk=True       # 하위 디렉토리 재귀 검색
default=        # keydata 없을 때 기본값
append=True     # 기존 파일에 추가
append_mode=    # "jsonl" | "array" | "object" | (미지정=자동감지)
jsonl=True      # JSONL 형식으로 읽기
dir_path=       # loc 기준 서브 경로 추가
```

---

## dir_path — 서브 경로

두 번째 positional 인자로 loc 기준 하위 경로 지정:

```python
# data/json/base/mytest/abc/test.json 에 저장
u.wjb({"k": "v"}, "test", "mytest/abc")
u.rjb("test", "mytest/abc")

# full-chain도 동일
use.read.json.base("test", "mytest/abc")

# /로 시작하면 절대 경로
u.rjb("test", "/external/path")
```

---

## keydata — 중첩 경로 접근

```python
u.rjb("config", keydata="user/email")
u.rjb("data",   keydata="items[0]/name")
u.rjb("cfg",    keydata="db/host", default="localhost")
```

---

## JSON Append 패턴

```python
# JSONL (줄 단위 JSON)
u.wjb({"date": "2026-01-01", "user": "Alice"}, "log", append=True, append_mode="jsonl")
u.rjb("log", jsonl=True)   # → [{"date":...}, ...]

# 배열에 항목 추가
u.wjb([{"id": 1}], "items")
u.wjb({"id": 2}, "items", append=True, append_mode="array")

# 오브젝트 병합
u.wjb({"a": 1}, "cfg")
u.wjb({"b": 2}, "cfg", append=True, append_mode="object")

# 자동 감지 (기존 파일 구조 보고 array/object 자동 선택)
u.wjb(new_item, "data", append=True)
```

## Mem Store — 프로세스 메모리 저장소

파일 없이 프로세스 내 dict에 저장. 세션 종료 시 소멸.  
포맷 prefix는 라우팅 관례일 뿐 — **모든 포맷이 동일한 `_STORE` 공유**.

```python
# 기본 CRUD
u.wjm(data, "key")          # 저장 (0.005ms)
u.rjm("key")                # 읽기
u.rjm("key", default=None)  # 없을 때 기본값
u.ujm({"b": 99}, "key")    # dict 병합 (기존 dict이면 update, 아니면 overwrite)
u.djm("key")                # 삭제
u.hjm("key")                # 존재 확인 → bool
u.ljm()                     # 전체 키 목록 → list

# 다른 포맷 prefix도 동일 동작 (같은 _STORE)
u.wym(data, "key")  / u.rym("key")   # yaml
u.wtm(text, "key")  / u.rtm("key")   # txt
u.wmm(md, "key")    / u.rmm("key")   # md
u.wcm(rows, "key")  / u.rcm("key")   # csv
u.wam(data, "key")  / u.ram("key")   # any
```

**실행 에디트 캐시 패턴:**
```python
# 무거운 연산 결과를 메모리에 캐시
u.wjm(expensive_result, "cache")

# 이후 재사용 — 파일 읽기 없음, 컨텍스트 재노출 없음
data = u.rjm("cache")
```

**ejm — emit (직렬화 전용, mem과 별개)**
```python
u.ejm({"k": "v"})            # → str (파일·저장 없음)
u.ejm({"k": "v"}, type="d")  # str/bytes → dict 역직렬화
u.ejm(data, type="b")        # → bytes
u.ejm(data, type="l")        # → list (줄 단위)
```

---

## Constraints

- `u.ejm()` — 직렬화 전용. 저장 없음, 반환값 `str`.
- mem store — 프로세스 재시작 시 소멸. 영속 필요 시 `wjb`/`wjc` 사용.
- `u.djb(name="*")` — 와일드카드 삭제 차단됨.
- `name` 미지정 → dumps 모드 (파일 생성 안 됨).
